# atoikura

家計簿ではなく、「あといくら使えるか」がわかるアプリ

atoikura は、
「何に使ったか」ではなく
「あといくら安心して使えるか」 を教えてくれる家計管理アプリです。

多くの家計簿アプリは、支出を記録し、分析し、反省させます。
しかし実際には、
• 明細を見るのが怖い
• グラフを見ても行動は変わらない
• 気づいたらまた使ってしまう

という人が多いのではないでしょうか。

atoikura はその逆のアプローチを取ります。

過去の支出ではなく、
これから使える金額を主役にする。

## コンセプト

普通の家計簿

```
何に使ったか
↓
反省
↓
改善
```

atoikura

```
あといくら使えるか
↓
安心して使える
↓
自然に貯まる
```

atoikuraのホーム画面では、
支出の明細や複雑なグラフは表示されません。

代わりに表示されるのはこれだけです。

```
今月あと使える金額
¥42,300

今日の安心ライン
¥1,410
```
